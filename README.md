

🏰 CastleApp (Echoes & Paths)
Discover European heritage through geospatial exploration.

📖 Overview
CastleApp is a mobile application designed to bridge the gap between modern technology and historical memory. Using the Europeana API, the app allows users to explore cultural landmarks and historical stories based on their real-time location.

Whether you are a traveler or a local history enthusiast, CastleApp transforms your surroundings into a living museum.

🚀 Key Features
Geospatial Search: Find historical locations within a 10km radius using advanced PostGIS queries.

Interactive Maps: Real-time navigation and landmark discovery powered by react-native-maps.

Curated History Feed: A dynamic, scrollable interface showcasing "Hero" locations and recent historical finds.

Detailed Heritage Views: Deep dives into historical data, including high-resolution imagery and rich metadata from Europeana.

🛠️ Tech Stack
Frontend
React Native & Expo: For a smooth, cross-platform mobile experience.

React Navigation: Complex stack and tab navigation.

Expo Location: Precise GPS handling and permission management.

Backend
Node.js & Express: Scalable API architecture.

PostgreSQL with PostGIS: Specialized for spatial and geographic data (optimized for "points of interest" queries).

Europeana API Integration: Seamless fetching of European cultural heritage data.
